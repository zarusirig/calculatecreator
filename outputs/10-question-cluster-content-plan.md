# Prioritized Question-Cluster Content Plan — ttcalculator.net

**Date:** 2026-07-10
**Source:** Ahrefs Keywords Explorer (US), `matching_terms`, filtered KD ≤ 20, volume ≥ 60
**Central entity:** TikTok creator monetization
**Strategic frame:** Outer Section (informational question hubs) → internal-links into → Core Section (calculators). Build topical authority around the money-making tools by *answering the questions people ask right before they'd use a calculator*.

---

## Read this first — the constraint that shapes everything

The site currently ranks for **0 organic keywords at DR 0** (both `tiktokcalculator.net` and `ttcalculator.net`). Most of the target pages in this plan **already exist** (guides, data pages) — they simply win nothing yet. So this is **not** a "write 30 new articles" plan. It is roughly:

- **~65% OPTIMIZE + INTERLINK** existing pages into tight, exact-match clusters
- **~35% CREATE** net-new pages for genuine exact-match gaps (a general guide exists, but the money query needs its own page + snippet)

Publishing net-new thin pages on a DR-0 site *lowers* the average quality of the map (non-quality pages drag quality pages down). Every item below is scoped to a **specific query intent with a specific page and a specific calculator to link into** — nothing speculative.

## Priority scoring model

Each cluster scored on four axes (1–5), priority = winnability-weighted:

- **Winnability** — KD + whether weak (low-DR) sites already rank top-10. Weighted heaviest for a DR-0 site.
- **Volume** — aggregate monthly US searches across the cluster.
- **Intent fit** — how directly the query sits next to a calculator (monetization) vs. off-context.
- **Speed** — OPTIMIZE existing page (fast) vs. CREATE (slow).

## Topical borders — explicitly DO NOT chase

These showed high volume but sit outside the monetization source context. Chasing them adds off-topic pages that dilute the map:

- **Buy engagement (black-hat):** buy tiktok followers (32k), buy tiktok views (4.8k), free tiktok views (7k) — spammy SERP, brand-risk, wrong audience.
- **Buy coins (reseller commercial):** tiktok coins buy / cheap / recharge / free (~15k combined) — reseller intent, not calculator intent.
- **Consumer shop:** tiktok shop promo code (31k), is tiktok shop safe (11k), customer service number (10k), tracking, return policy — buyer intent, no calculator path.
- **App how-to:** how to turn off/see profile views (7.5k+), how to hide followers — utility, not monetization.
- **Non-English coin terms:** tiktok coins recargar / gratis / monedas — different market/language.

## Why this is "content, not tools" — and the caveat

Every winnable query above is a **question**, not a tool request. The tool-shaped demand ("engagement rate calculator", "money calculator") is already built and totals ~2k/mo; the question demand totals **~120k/mo** across P1–P2. So: build content.

But remember the gating finding — **none of this ranks until DR ≠ 0.** Ship 3–4 of these optimizations, then spend equal effort on 3–5 genuine editorial links (original data studies from your RPM/gift tables pitched to creator-economy press). Content + links together; content alone against DR 0 will repeat the last 12 months.
